Map Overlays


PAMGUARD uses a system of overlays to display data from detectors and other PAMGUARD modules on the map. Which overlay options are available will depend on which modules are currently loaded.

Turning overlays on and off

The easiest and quickest way to turn overlays on and off is from the map pop up menu, accessed by clicking with the right mouse button anywhere on the map.



You may select or de-select each overlay item. Selected items will appear in the map and be listed in the map key.

For more advanced options, use the topmost item on the pop-up menu "Plot Overlay Options..." to access the overlay options dialog.



This allows the same selection of items as the pop-up menu, but also offers additional options to specify for how long each type of data should be displayed.
